# Marmara Not

A web-based platform concept developed to enable university students
to openly share and access lecture notes, organized by faculty and department.

## Overview
Marmara Not was designed as a website project to provide a centralized
and structured environment where students can share academic notes
in an open and accessible way.

The platform aims to categorize content by faculty and department,
making it easier for students to find relevant materials.

## Features (Planned)
- Faculty and department-based content organization
- Open note sharing between students
- Simple and accessible web interface

## Technical Stack
- Python
- Web-based application architecture
